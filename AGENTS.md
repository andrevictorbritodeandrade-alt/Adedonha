# Style Guide - Arena de Jogos

Para manter a consistência visual de todos os avatares e imagens do projeto, utilize o seguinte Prompt Mestre:

## Estilo Master
"Estilo animação 3D realista (estilo Pixar/Disney moderno), personagem negro com traços detalhados, iluminação cinematográfica suave, cores vibrantes, fundo levemente desfocado (bokeh), enquadramento de meio corpo, centralizado."

## Exemplos de Uso:

- **Xadrez:** "Personagem negro, mestre de xadrez, expressão focada e estratégica, peças de xadrez holográficas flutuando ao redor, [ESTILO MASTER]."
- **Adedonha:** "Professor negro, mestre das palavras, cercado por letras mágicas flutuantes, [ESTILO MASTER]."
- **Perguntados:** "Gênio negro, aura de sabedoria, segurando uma esfera de conhecimento brilhante, [ESTILO MASTER]."
- **Detetive:** "Investigador negro, estilo detetive noir moderno, segurando uma lupa tecnológica, [ESTILO MASTER]."
